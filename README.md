## Installation

Clone the project

```bash
  git clone https://github.com/MathieuMarthy/Medimate-BD
  cd Medimate-BD/
```

Install requirements

```bash
  pip install -r requirements.txt
```

Start the script

```bash
  python3 main.py
```
